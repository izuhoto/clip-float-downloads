# ClipFloat Downloads

This repository hosts the distribution site content for **ClipFloat**, a screen capture app for macOS.

ClipFloat is designed for more than just taking screenshots. After capture, you can continue editing, organizing, and reusing results in the same workflow.

## Platform

- macOS 14 (Sonoma) or later

## Key Features

- Fast capture of the exact area you need
- In-place editing with annotations and text
- OCR-based text extraction from images
- Capture history management for quick reuse
