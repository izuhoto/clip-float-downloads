# Clip Float v1.0.2

2026-05-20

v1.0.2では、キャプチャ操作と履歴操作の使い勝手・安定性を中心に改善し、あわせてリリース/ビルド周りのメンテナンスも実施しました。

## 主な変更
- プレビューからのドラッグ書き出しが環境によって不安定になるケースに対して、フォールバック経路を追加。
- キャプチャ時のガイダンス/HUD表示を必要に応じて隠せるようにし、領域選択時の視認性を向上。
- ダークモードで歯車アイコンが見えづらい問題を改善。

## 不具合修正
- 履歴一覧で Delete ショートカットによる削除が動作しない回帰を修正。
- プレビュー書き出し時のドラッグ操作を修正（アクセシビリティ系の挙動を含む）。
- Space キーによる HUD トグル挙動とヒント文言の整合性を修正。
- キャプチャ/HUD連携や関連UI挙動の追加修正を反映。

## 品質・保守性
- Delete コマンド判定まわりのヘルパー命名を整理し、意図を明確化。
- CaptureSessionController と HistoryBrowserCoordinator のテストを拡充・更新。
- Makefile/README など、リリース手順・ビルド運用に関するメンテナンスを実施。

# Clip Float v1.0.1

2026-05-19

v1.0.1 では、履歴閲覧まわりの安定性と使いやすさ、ならびに配布品質の向上に注力しました。

## 主な改善
- プレビューウィンドウ右上の書類・設定アイコンの視認性を改善し、白背景でも見やすくしました。
- 履歴ウィンドウのサイズ保存・復元ロジックを見直し、極端に小さい状態や使いにくい状態への復元を防止しました。
- 保持期間クリーンアップ周辺の内部実装を改善し、堅牢性を高めました。

## 不具合修正・安定性向上
- ウィンドウ移動・復元時にキャプチャ領域サイズが 1px ずれることがある問題を修正しました。
- 保持期間設定変更時の履歴・ゴミ箱クリーンアップの追従性を改善しました。
- ゴミ箱のソート順を保持するようにし、再表示時の一貫性を向上しました。

# Clip Float v1.0.0

2026-05-15

Clip Float の初回リリースです。
画面キャプチャ、キャプチャ履歴管理、そしてキャプチャ画像編集の基本機能を利用できます。

## 主な機能
- 画面キャプチャの実行
- 履歴の保存と再表示
- 画像編集
- 注釈の追加・編集
